第1条（定義）

  1. 「月次運営権」とは、指定施設の指定月における宿泊売上相当額の宿泊クレジット受取権をいいます。
  2. 「宿泊クレジット」とは、SOLUNA全施設の宿泊料金に充当できる電子価値をいいます。
  3. 「対象月」とは、利用者が購入時に指定した暦月（本契約では{{YEAR}}年{{MONTH}}月）をいいます。
  4. 「対象施設」とは、利用者が購入時に指定した施設（本契約では{{PROPERTY_NAME}}）をいいます。

第2条（本契約の性質）

  1. 本契約は、宿泊クレジットの前払購入に関するサービス利用契約です。
  2. 本契約は、不動産の所有権・共有持分・事業への出資・投資収益の分配を約束するものではありません。
  3. 当社は利用者に対して金銭の返還義務を負いません。クレジットのみを対価として付与します。
  4. 宿泊クレジットは、資金決済法における前払式支払手段として発行するものではなく、サービスの対価として付与される施設内利用権原です。

第3条（契約の成立）

  1. 利用者が当社の指定する決済手段により購入代金の支払いを完了した時点で、本契約が成立します。
  2. 各対象月・各対象施設につき購入できる利用者は1名に限られます。購入完了後は他の利用者への販売を行いません。

第4条（宿泊クレジットの付与）

  1. 当社は、対象月翌月10日（土日祝日の場合は翌営業日）を目途に、対象月における対象施設の宿泊売上額（税抜・OTA手数料控除後）を宿泊クレジットとして利用者に付与します。
  2. 売上額の確定に時間を要する場合、付与時期を延期することがあります。その場合は利用者にメールで通知します。
  3. 付与するクレジット額は実際の売上に基づくものであり、購入価格・過去実績等を将来にわたって保証するものではありません。
  4. 売上の計算方法：Airbnb・予約サイト等への掲載売上合計 ÷ 宿泊料のみ（食事・アクティビティ等除く）。

第5条（不可抗力による休業）

  1. 天災地変、行政規制、感染症の拡大、設備の重大な故障その他当社の責に帰すことができない事由により、対象月中に対象施設が全期間または一部期間休業した場合、当社は以下の対応をとります。
    • 全期間休業: 購入価格相当額の宿泊クレジットを付与
    • 一部期間休業: 実際の売上に応じたクレジットを付与（最低保証なし）
  2. 前項の場合、当社はそれ以上の損害賠償・補償の義務を負いません。

第6条（キャンセル）

  1. 対象月開始の14日前（{{CANCEL_DEADLINE}}）までは、当社への申し出によりキャンセルが可能です。この場合、購入価格から処理手数料（購入価格の10%）を控除した金額相当の宿泊クレジットを付与します（現金返金は行いません）。
  2. 対象月開始の14日前以降のキャンセルは原則として受け付けません。
  3. キャンセルの申し出は info@enablerdao.com への電子メールにより行うものとします。

第7条（宿泊クレジットの利用条件）

  1. 付与された宿泊クレジットは、SOLUNA全施設（弟子屈・熱海等）の宿泊料金に充当できます。
  2. 有効期限は付与日から2年間です。期限経過後のクレジットは消滅します。
  3. 宿泊クレジットの現金への換金はできません。
  4. 宿泊クレジットの第三者への譲渡・売却はできません。ただし当社が別途認めた場合を除きます。
  5. 宿泊クレジットは宿泊料金にのみ使用でき、物品・食事・アクティビティ等には使用できません。

第8条（個人情報の取扱い）

当社は、本契約に関連して取得した利用者の個人情報を、当社プライバシーポリシー（https://solun.art/privacy.html）に従い適切に管理します。

第9条（免責の上限）

本契約に関して当社が負う損害賠償の上限は、購入価格相当の宿泊クレジット付与をもって上限とします。当社は間接損害・逸失利益について責任を負いません。

第10条（準拠法・管轄）

本契約は日本法に準拠します。本契約に関する紛争の第一審の専属的合意管轄裁判所は東京地方裁判所とします。
