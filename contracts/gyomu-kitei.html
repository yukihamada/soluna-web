第1章　総則
第1条（目的）

本規程は、株式会社イネブラ（以下「当社」という。）が行う小規模不動産特定共同事業（不動産特定共同事業法〔以下「法」という。〕第2条第11項に規定する事業をいう。）の業務を適正かつ効率的に遂行するため、その基本的事項を定めることを目的とする。

第2条（定義）

本規程において使用する用語の定義は、次のとおりとする。

  1. 「小規模不動産特定共同事業」とは、法第2条第11項に規定する事業（1件の不動産特定共同事業契約に係る出資の総額が1億円を超えず、かつ各出資者の出資額が100万円を超えない事業）をいう。
  2. 「不動産特定共同事業契約」とは、法第2条第3項に規定する契約をいい、本事業においては任意組合契約（民法第667条）を採用する。
  3. 「出資者」とは、当社と不動産特定共同事業契約を締結し、出資を行う者をいう。
  4. 「業務管理者」とは、法第2条第13項に規定する者であって、当社に設置する者をいう。
  5. 「提携宅建業者」とは、当社が不動産取引業務を委託する、宅地建物取引業法（以下「宅建業法」という。）に基づく免許を有する業者をいう。
  6. 「対象不動産」とは、各事業において取得・運営の対象となる宿泊施設用不動産をいう。
第3条（適用範囲）

本規程は、当社が法に基づき都道府県知事の登録を受けて行うすべての小規模不動産特定共同事業に適用する。

第2章　事業の実施
第4条（事業の内容）
  1. 当社は、宿泊施設用不動産を対象とした小規模不動産特定共同事業を行う。各事業の概要は以下のとおりとする。
    1. 対象地域：日本全国（当初は北海道・静岡県・香川県・島根県等を予定）
    2. 事業内容：空き家・古民家等の取得、リノベーション、宿泊施設としての運営および利益分配
    3. 1件あたり出資総額：1億円以下
    4. 1人あたり出資額：100万円以下
    5. 出資者への提供価値：年間宿泊利用権（20泊/年を上限として各事業で設定）および運営収益の分配
  2. 当社は、各事業について個別の事業計画書を策定し、出資者への開示に備えるものとする。
第5条（物件選定基準）
  1. 当社は、対象不動産の選定に際し、以下の基準を充足することを確認するものとする。
    1. 建物の構造・耐震性：昭和56年以降の新耐震基準適合、または耐震診断による安全性の確認
    2. 権利関係：抵当権・差押等の負担がないこと（または取得時に解消できること）
    3. 法令上の制限：建築基準法・旅館業法・消防法等の適法性
    4. 収益性：リノベーション後の想定稼働率・売上を試算し、出資者への分配可能性を確認
    5. 環境リスク：土壌汚染・アスベスト・PCB等の有害物質がないこと
  2. 当社は、取得前に現地調査を実施し、その結果を記録・保存するものとする（別表1参照）。
  3. 重大なリスクが確認された場合、当社は取得を中止し、その旨を記録する。
第6条（宅地建物取引業者との連携）
  1. 当社は、不動産取引に係る業務（売買・賃貸の仲介、重要事項説明等）については、提携宅地建物取引業者〔●●〕（免許番号：〔●●〕）に業務委託し、当該業者が宅建業法に基づき適正に処理する。
  2. 業務委託の内容および各者の役割分担は次のとおりとする。
    業務担当
    不動産売買の仲介・媒介提携宅建業者〔●●〕
    重要事項説明書の作成・交付提携宅建業者〔●●〕
    不動産特定共同事業契約の締結・管理当社
    出資者の募集・勧誘当社
    施設の運営管理・宿泊予約管理当社
    収益分配・報告当社
  3. 当社は、提携宅建業者との間で書面による業務委託契約を締結し、これを保存するものとする。
第7条（事業収支計画の策定）
  1. 当社は、各事業の開始前に以下を含む事業収支計画を策定する。
    1. 取得費・リノベーション費の見積もり
    2. 年間稼働率・売上の想定（過去の類似物件実績を参照）
    3. 運営費用（清掃費・OTA手数料・消耗品費等）の見積もり
    4. 出資者への分配予定額および分配時期
    5. リスク要因（空室リスク・修繕リスク・自然災害等）
  2. 事業収支計画は、出資者への契約締結前書面（法第24条）に添付して開示するものとする。
第3章　募集・勧誘
第8条（勧誘原則）
  1. 当社は、出資者の勧誘に際し、以下の原則を遵守する。
    1. 誠実義務：出資者の利益を最優先し、誠実かつ公正に業務を行う
    2. 適合性原則：出資者の知識・経験・財産状況・目的に照らして不適切な勧誘を行わない
    3. 説明義務：リスク（元本損失の可能性、流動性リスク等）を明確に説明する
    4. 誇大広告禁止：法第21条に基づき、著しく事実と相違する表示または誤認させる表示を行わない
  2. 当社は、次に掲げる行為を行ってはならない。
    1. 断定的判断の提供（「必ず利益が出る」等の断言）
    2. 不確実な事項について確実であるかのような誤解を生じさせる表示
    3. 不招請勧誘（意思表示のない者への訪問・電話による勧誘）
第9条（告知書面の交付）
  1. 当社は、出資者に対して法第23条に基づく書面（以下「告知書面」という。）を交付する。
  2. 告知書面には、以下の事項を記載する。
    1. 当社の商号、住所、登録番号
    2. 対象不動産の概要（所在地・面積・構造・権利形態）
    3. 事業の仕組み・出資者の権利・義務の概要
    4. 出資金の用途・返還に関する事項
    5. リスクに関する事項（元本保証なし等）
    6. クーリングオフに関する事項
  3. 告知書面はウェブサイト上での電磁的方法による提供も可とするが、出資者の承諾を得た場合に限る。
第10条（勧誘方法の制限）
  1. 当社の勧誘方法は、以下に限定する。
    1. 当社ウェブサイト（solun.art）上での情報提供・申込受付
    2. 電子メールによる案内（登録会員への情報提供）
    3. 説明会・セミナーにおける対面説明（事前申込制）
    4. 既存出資者からの紹介（紹介料の支払いがある場合は開示）
  2. 不特定多数を対象とした第三者媒体への広告掲載は、業務管理者の事前承認を得て行う。
第11条（出資者の確認）
  1. 当社は、出資者との契約締結前に以下を確認する。
    1. 本人確認：犯罪収益移転防止法に基づく本人確認（運転免許証・マイナンバーカード等）
    2. 反社会的勢力との関係がないことの確認
    3. 出資目的・投資経験・財産状況の把握（適合性確認シートにより記録）
  2. 未成年者・成年後見人・被保佐人等の行為制限のある者との契約は、法定代理人の同意等が必要である。
第4章　契約締結
第12条（契約締結前書面の交付）
  1. 当社は、法第24条第1項に基づき、出資者と不動産特定共同事業契約を締結しようとするときは、あらかじめ契約締結前書面を交付する。
  2. 契約締結前書面には、法令に定める事項（対象不動産の表示・事業収支計画・クーリングオフ・解除条件等）を記載する。
  3. 書面の交付から契約締結まで、原則として72時間以上の熟慮期間を設ける。
第13条（契約締結時書面の交付）
  1. 当社は、法第24条第2項に基づき、契約締結後遅滞なく契約締結時書面を交付する。
  2. 契約締結時書面は、電磁的方法による提供も可とするが、出資者の承諾を要する。
第14条（クーリングオフ）
  1. 出資者は、法に定めるクーリングオフ期間（契約締結時書面受領日から8日間）内に、書面または電磁的記録により契約を解除することができる。
  2. クーリングオフが行使された場合、当社は出資金を速やかに（原則5営業日以内に）全額返還する。
  3. クーリングオフの行使に際し、出資者に損害賠償その他の費用を請求しない。
第15条（契約の記録・保存）
  1. 当社は、不動産特定共同事業契約書・締結前書面・締結時書面・クーリングオフ通知等の書類を、契約終了後10年間保存する。
  2. 電磁的記録による保存も可とするが、改ざん不可能な形式（PDF/A等）で行う。
第5章　財産管理
第16条（分別管理）
  1. 当社は、出資者から受け入れた出資金を、当社の固有財産および他の事業の出資金と明確に分別して管理する。
  2. 分別管理の方法：各事業専用の銀行口座を開設し、当該口座においてのみ出資金の受入・管理を行う。
  3. 出資金は、対象不動産の取得・リノベーション費用のみに充当し、当社の運転資金等には使用しない。
第17条（信託・供託の方針）
  1. 当社は、出資金の安全管理を確保するため、必要に応じて信託銀行への信託または法務局への供託を検討する。
  2. 現時点では分別口座管理（第16条）を基本とし、事業規模の拡大に伴い信託管理への移行を検討する。
第18条（運営費用の区分）
  1. 収益分配の計算にあたり、控除する費用は以下のとおりとする（間接費は控除しない）。
    1. OTA（宿泊予約サイト）手数料
    2. 清掃費・消耗品費等の直接運営費
    3. リノベーション後に発生した設備修繕費（30万円以下の小修繕）
    4. 各事業専用口座の振込手数料
  2. 管理費・人件費等の間接費は当社が負担し、出資者の分配計算から控除しない。
第6章　運営報告
第19条（定期報告）
  1. 当社は、各出資者に対して四半期ごとに運営報告書を交付する。
  2. 運営報告書の交付方法：電子メール（出資者が電磁的方法を承諾した場合）または書面郵送。
  3. 報告書の交付期限：各四半期終了後30日以内。
第20条（収支報告書の作成）
  1. 当社は、各事業の期間終了後60日以内に、以下を含む収支報告書を作成・交付する。
    1. 宿泊売上の内訳（月別・チャネル別）
    2. 控除費用の内訳（OTA手数料・清掃費等）
    3. 分配額の計算根拠
    4. 未分配残高がある場合の処理方針
  2. 出資者は、法第18条（法第63条の10準用）に基づき収支計算書の閲覧を請求できる。
第21条（情報開示）
  1. 当社は、出資者から個別の情報提供要求があった場合、合理的な範囲内で速やかに（原則10営業日以内に）対応する。
  2. 以下の情報は、当社ウェブサイト上で随時開示する。
    1. 登録番号・登録年月日
    2. 業務規程（本規程）
    3. 不動産特定共同事業契約約款
    4. 苦情相談窓口
第7章　業務管理者
第22条（業務管理者の設置）
  1. 当社は、法第63条の5に基づき、主たる事務所（東京都港区三田）に業務管理者を1名以上設置する。
  2. 業務管理者：〔●●〕（不動産特定共同事業業務管理者講習修了番号：〔●●〕）
  3. 業務管理者が欠けた場合、当社は2週間以内に後任者を確保するための措置を講じる。
第23条（業務管理者の職務）
  1. 業務管理者は、以下の職務を担当する。
    1. 法令遵守状況の確認・指導
    2. 契約締結前書面・契約書の内容審査
    3. 勧誘・募集方法の適切性確認
    4. 出資者からの苦情・相談への対応統括
    5. 業務規程の見直し・改定の起案
    6. 定期報告書・収支報告書の内容確認
第24条（業務管理者の監督）
  1. 代表取締役は、業務管理者が適正に職務を遂行しているかを定期的（年2回以上）に確認する。
  2. 業務管理者は、重大な法令違反のおそれがある場合、直ちに代表取締役に報告する義務を負う。
第8章　苦情処理・紛争解決
第25条（苦情受付窓口）
  1. 当社は、出資者からの苦情・相談を受け付けるための窓口を設置する。
    苦情・相談受付窓口
    株式会社イネブラ　業務管理者宛
    メール：info@enablerdao.com
    受付時間：平日 10:00〜17:00（土日祝・年末年始を除く）
第26条（対応手順）
  1. 当社は、苦情・相談を受け付けた場合、以下の手順で対応する。
    1. 受付日から7営業日以内に、受付確認および初回回答（または調査中の旨の通知）を行う
    2. 事実関係の調査・確認（最大30営業日）
    3. 解決策の提示・実施
    4. 対応結果の記録（台帳への記入・10年保存）
  2. 当社は、正当な理由なく苦情・相談への対応を拒否してはならない。
第27条（紛争解決機関）
  1. 当社は、出資者との紛争が当事者間で解決できない場合、以下の指定ADR機関の利用を出資者に案内する。
    1. 公益財団法人 不動産流通推進センター（苦情・相談窓口）
    2. 一般社団法人 不動産証券化協会（ADR）
    3. 法務省・裁判外紛争解決手続（ADR）認定機関
  2. 当社は、ADR機関への対応に誠実に協力する義務を負う。
第9章　記録・保存
第28条（帳簿の作成・保存）
  1. 当社は、不動産特定共同事業ごとに以下の帳簿を作成し、事業終了後10年間保存する。
    帳簿の種類主な記載事項保存期間
    出資者台帳氏名・住所・出資額・出資日・本人確認記録10年
    契約台帳契約番号・締結日・対象物件・契約金額10年
    収支台帳売上・費用・分配額の月次記録10年
    苦情台帳受付日・内容・対応経緯・解決結果10年
    物件管理台帳物件情報・取得価格・リノベ費・稼働状況10年
第29条（電磁的記録）
  1. 帳簿の保存は電磁的記録（クラウドストレージ等）によることができる。
  2. 電磁的記録による保存の場合、改ざん防止措置（アクセスログの保存・定期バックアップ）を講じる。
  3. システム障害等により電磁的記録が消失した場合の復旧手順を定め、別途文書化する。
第10章　改廃
第30条（規程の改廃）
  1. 本規程の制定・改廃は、代表取締役の承認を得て行う。
  2. 法令の改正等により本規程の変更が必要となった場合、業務管理者は速やかに改定案を起案し、代表取締役に提出する。
  3. 改廃の履歴は本規程冒頭の改定履歴テーブルに記録する。
  4. 改廃後の規程は、出資者に対してウェブサイト上で開示する。

消防法（自動火災報知設備・誘導灯等）の適合確認
農地転用・開発許可等の要否確認
道路・河川等の法令制限確認

4. 収益性確認

周辺類似物件のOTA（AirBnB・楽天トラベル等）稼働率・単価調査
事業収支計画（3ケース：楽観・標準・悲観）の作成
投資回収期間の算定
出資者への分配可能額の試算（直接運営費控除後）

5. 環境・地域確認

洪水・土砂災害ハザードマップの確認
地域住民・自治会との関係（民泊への理解度）確認
最寄り交通機関・アクセス状況の確認
近隣施設（スーパー・温泉・観光地等）の確認
別表2　苦情処理フロー
1
苦情・相談の受付
メール／電話で受付。受付日時・内容・担当者を苦情台帳に記録。
↓
2
初回回答（7営業日以内）
受付確認メールを送付。調査が必要な場合はその旨と目処を連絡。
↓
3
事実関係の調査（最大30営業日）
業務管理者が関係書類・関係者へのヒアリングを実施。
↓
4
解決策の提示・実施
調査結果を出資者に説明し、対応策を提示。合意が得られた場合は実施。
↓
5
解決確認・記録
出資者の納得を確認し、苦情台帳にクローズ日・解決内容を記録。
↓（合意に至らない場合）
6
ADR機関へのあっせん案内
不動産流通推進センター等のADR機関を出資者に案内し、協力する。
