KUMAUSHI BASE · 北海道 弟子屈 熊牛原野 · 43°22′N 144°30′E

コンテナ12台 × 非対称コの字。草原の真ん中に、全部ある。

熊牛原野の尾根に立つと、半径2kmに民家がない。騒音クレームがない。消防からの制限もない。北には阿寒岳、東には屈斜路湖、南は太平洋まで続く丘陵地帯。ただ風と草原と、遠い山並みだけがある。

ここにコンテナ12台を並べる。中庭には電動折りたたみPTFE膜（120m²）。朝は柔術道場、昼はウェルネス、夜はパーティー会場。同じ空間が1時間で別の場所に変わる。南端には温水インフィニティプール——水面が草原と地平線に溶けていく。

2FにはDJブース。Pioneer CDJ-3000、Funktion-One音響。爆音でいい。隣人がいないから。バーは1F——ヒバリ、山崎、タップビール3種。シーシャはルーフデッキか中庭。ビリヤード9ft、電子ダーツ、卓球、ボクシングサンドバッグ。プロジェクションは北壁12m幅を使う。

サウナ→水風呂→温水プール→外気浴のサーキットは世界中どのスパにも負けない。熊の声が聞こえる場所で、天の川を見上げながらととのう。冬は雪原がプライベートコースになる。スノーモービル、スノーバイク。夏は道東の尾根道をハーレーで走ってくる。スバル アウトバックでも。EV充電器は太陽光直結だ。

Tesla Powerwall 3 × 2台（合計40kWh）と太陽光35kWが電力を自給する。薪ボイラーが全館床暖房とプールを温める。Starlinkがインターネットを提供する。電柱ゼロ。電力会社ゼロ。それでも何も不足しない。

FAQ · よくある質問

よくある、6つの疑問。

1口¥480万の内訳はどうなっていますか？ +
総事業費¥9,600万（土地・コンテナ12台・プール・DJシステム・音響・バー設備・エンタメ機器・膜・全工事費＋バッファ¥2,000万）を20口で分割します。1口¥480万。20口（全口購入時）で10コンテナ施設の持分会員権登録を取得します。別途、年間管理費¥35万（管理・メンテナンス・光熱費・予約代行含む）が発生します。
年30泊の滞在権は、どのように使えますか？ +
SOLUNAのオーナー専用予約システムから先着順で予約できます。シーズンの制限はありませんが、繁忙期（GW・夏・年末年始）は抽選となる場合があります。使わなかった泊数は外部貸出に回り、収益配当として還元されます。
収支シミュレーションの根拠は何ですか？ +
宿泊単価¥80,000は弟子屈エリアの高級プライベートキャビン（2〜6名）の市場単価を参考にしています。稼働率は弟子屈町の既存施設（屈斜路湖サウナクラブ等）の実績を基準に、保守40%・中間60%・楽観80%の3シナリオを設定しました。管理費¥35万/口は固定で控除します。
法的にどのような権利を持ちますか？ +
SOLUNA利用権型会員権として発行します。登記完了後、オーナーは法的に土地・建物の1/10を所有します。担当者によるビデオ通話での購入前確認を実施したうえで契約を締結します。持分は他のオーナーへの譲渡も可能（協議による）です。
コンテナは北海道の冬（−30℃）に耐えられますか？ +
外壁に閉セル型SPF（発泡ウレタン）を100mm吹付。熱伝導率λ=0.022 W/m·K、R値≥4.5。コンテナ鉄板の結露を外側から完全にシャットアウト。吹付後にオーナーが石灰漆喰を直接手塗り（石灰系プライマー1回→漆喰）。ひびが入ったら年次Work Partyで塗り直す前提の設計。薪ボイラー+温水床暖房で室内は全館暖房。−30℃対応。
柔術・瞑想・断食リトリートはどのように提供されますか？ +
熊牛原野の中庭（コンテナコの字）には白いEVAマットを常設します。SOLUNAは世界クラスのブラジリアン柔術インストラクター、瞑想ガイド、断食サポートの専門家と提携し、プログラムを提供します。オーナーは自己使用滞在時に優先参加できます。宿泊ゲスト向けの有料プログラムは収益に算入されます。
Work Partyとは何ですか？誰でも参加できますか？ +
Work Partyのメインは外壁漆喰塗り。コンテナ設置後にSPF（発泡ウレタン）100mmを吹付済みの状態で来ていただき、オーナー全員で漆喰を直接手塗り。石灰プライマー1回塗ってから塗り始めます。割れたら次のWork Partyでまた塗る——その繰り返しがコミュニティの年次行事になります。定員100名・参加費無料。2026年9月20〜22日。最終日試泊付き。
NFTオーナーパスとは何ですか？どこで使えますか？ +
NFTオーナーパスはNFCタグ（カードまたはシール）に書き込まれた固有IDです。施設のドアロック・サウナ・バレルバスをタッチするだけで解錠できます。スマートフォンアプリ（KAGI）と連携し、施設設備の遠隔操作・予約確認・滞在ログの確認が可能です。GETFREEプロモコードをお持ちの方は無料で取得できます（→ solun.art/getfree）。
KAGIアプリとは何ですか？ +
KAGIはSOLUNA施設専用のスマートホームアプリ（iOS）です。NFCタグとBluetoothを使ったドアの解錠、Philips Hue照明の調光・カラー変更、サウナや水風呂の温度確認、Starlinkの接続状態確認、施設の予約管理ができます。App Storeで「KAGI」と検索してダウンロードできます。CABIN CLUBオーナーには施設IDとパスコードを発行します。
