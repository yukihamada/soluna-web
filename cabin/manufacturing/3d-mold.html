SECTION 01

システム概要 — 3部品でビスゼロ接合

合板でモールドボックスを作り、3D印刷したインサートで接合プロファイルを型付け。培養後はTPUクリップでパネル同士をロック。ネジ1本も使わない。

システム全体図 — 型→培養→接合の流れ
なぜ合板ボックス + 3Dプリントインサートなのか

菌糸体パネルは600×600mm = 0.36m²。これを丸ごと3Dプリントしようとすると家庭用プリンタ（床面積220×220mm）では印刷できない。
合板で箱を作って、接合プロファイル（舌・溝）の部分だけインサートを印刷してはめ込む。これで任意サイズのパネルが量産できる。

型インサートは1セット印刷すれば何度でも再利用。培養ごとに型を解体→再組み立てするだけ。
PLAトングインサート + TPU Cクリップ。ビスゼロ。
溝にインサートをはめ込み、クリップで固定。工具不要。
接合完了。抜け止めは台形断面で機械的ロック。
SECTION 02

接合プロファイル詳細 — スケール1:1

台形断面（ダブテール類似）。抜け方向に対して抵抗があり、スライド方向には素直に動く。培養収縮（約2%）を見越した寸法公差を設定。

舌（tongue）断面 — 水平カット
溝（groove）断面 — 水平カット
C-クリップ断面（TPU）— スケール 1:1
TPU 95A（硬質ゴム弾性） 全幅: 104mm（各パネル30mm噛む） 高さ: 120mm（パネル1/8スパン） 壁厚: 4mm（スナップ嵌合に必要） 間隔: 300mm@1本（2クリップ/パネル高）

印刷パーツ詳細 — 3種類

1
型インサート（TONGUE / GROOVE）
材料: PLA / 印刷時間: 4h / コスト: ¥120/本 / 用途: 型枠の中に固定して接合面を成形
パラメータ値
印刷方向縦（長辺がZ軸方向）
サポート不要（テーパー5°）
インフィル40%（強度重視）
レイヤー厚0.2mm
外壁数4ペリメータ
離型処理離型スプレーまたは蜜蝋
使い方
合板ボックスの内壁にM3ボルト2本で固定。培養後に型を解体してインサートを引き抜く。

注意
舌側と溝側は別々のインサートが必要。1セット（舌×2 + 溝×2）で1枚のパネルが作れる。長辺方向に複数連結可。
2
C-クリップ（PANEL CLIP）
材料: TPU 95A / 印刷時間: 3h / コスト: ¥80/個 / 用途: パネル接合部にスナップ嵌合
パラメータ値
印刷方向クリップ開口部が上向き
サポート不要
インフィル25%（弾性確保）
レイヤー厚0.3mm（TPUは厚め）
外壁数3ペリメータ
印刷速度25mm/s（TPUは遅く）
取り付け手順
1. パネルの舌を溝に差し込む（スライド方向）
2. Cクリップを関節部に当てて押す
3. 「カチッ」とスナップ嵌合
4. 完了。工具不要。

取り外し
マイナスドライバー1本でこじれば外れる。分解・移設が可能。
3
コーナーコネクター（90°）
材料: TPU 95A / 印刷時間: 6h / コスト: ¥150/個 / 用途: 壁コーナーの直角接合
コーナーコネクター（平面）
設計ポイント
コーナーコネクターはC-クリップの90°版。両方向のパネルにスナップ嵌合する。

注意
TPUは90°形状の印刷が難しい。ブリッジ部分でサポート不要になるよう丸みを付けること。

バリエーション
45°コネクター（六角形レイアウト用）も同じOpenSCADパラメータで生成可能。
SECTION 04

OpenSCADコード — パラメトリック設計

OpenSCADは無料のパラメトリック3Dモデラー。このコードをコピーして貼り付ければSTLが生成できる。パラメータを変えるだけで全パーツが連動して変わる。

使い方
1. openscad.org からOpenSCADをダウンロード（無料）
2. 下記コードをコピーして新規ファイルに貼り付け
3. F5でプレビュー、F6でレンダリング、F7でSTLエクスポート
4. 最後の行のコメントアウトを変えて印刷したいパーツを切り替える 
// ============================================================
// SOLUNA Mycelium Panel System — 3D Print Molds v1.0
// OpenSCAD parametric design
// License: MIT / SOLUNA × Enabler Inc. 2026
// ============================================================

// ---- Global Parameters (change these to resize everything) ----
panel_t   = 100;   // Panel thickness (mm)
tongue_w  = 30;    // Tongue width at tip (mm)
tongue_d  = 15;    // Tongue protrusion depth (mm)
tongue_h  = 80;    // Tongue height (mm)
taper     = 5;     // Draft angle (degrees)
clearance = 1.0;   // Groove/tongue clearance (mm)
clip_grip = 30;    // How far clip grips each panel (mm)
clip_wall = 4;     // Clip wall thickness (mm)
fillet_r  = 2;     // Fillet radius (mm)
insert_l  = 200;   // Mold insert length (for 600mm: print 3x)
$fn       = 32;    // Curve resolution

// ---- Helper: Tongue profile (2D) ----
module tongue_profile() {
  w_base = tongue_w + 2 * tongue_d * tan(taper);
  polygon([[0, 0],
             [tongue_d, -(tongue_w / 2)],
             [tongue_d,  (tongue_w / 2)],
             [0, w_base / 2],
             [0, -w_base / 2]]);
}

// ---- Helper: Groove profile (2D, tongue + clearance) ----
module groove_profile() {
  c = clearance;
  w_base = tongue_w + 2 * tongue_d * tan(taper);
  polygon([[0, 0],
             [-(tongue_d + c),   -(tongue_w / 2 + c)],
             [-(tongue_d + c),    (tongue_w / 2 + c)],
             [0,  w_base / 2 + c],
             [0, -w_base / 2 - c]]);
}

// ============================================================
// PART 1A: Tongue Insert (glue inside mold box edge)
// Print in PLA. Attach to mold, remove after cultivation.
// ============================================================
module tongue_insert(length = insert_l) {
  back_w = tongue_w + 2 * tongue_d * tan(taper) + 6;
  difference() {
    union() {
      // Backing plate (attaches to mold wall)
      cube([6, back_w, length]);
      // Tongue shape extruded along length
      translate([6, back_w / 2, 0])
        rotate([90, 0, 90])
          linear_extrude(length)
            tongue_profile();
    }
    // M3 mounting holes x 3 along length
    for (i = [length/6, length/2, length*5/6]) {
      translate([3, back_w / 2, i])
        rotate([0, 90, 0])
          cylinder(h = 7, r = 1.75);
    }
  }
}

// ============================================================
// PART 1B: Groove Insert
// ============================================================
module groove_insert(length = insert_l) {
  w_total = tongue_w + 2 * tongue_d * tan(taper) + clearance * 2 + 10;
  difference() {
    // Backing plate
    cube([tongue_d + clearance + 6, w_total, length]);
    // Subtract groove cavity
    translate([0, w_total / 2, -1])
      rotate([90, 0, 90])
        linear_extrude(length + 2)
          groove_profile();
    // M3 mounting holes
    for (i = [length/6, length/2, length*5/6]) {
      translate([tongue_d + clearance + 3, w_total / 2, i])
        rotate([0, 90, 0])
          cylinder(h = 7, r = 1.75);
    }
  }
}

// ============================================================
// PART 2: Panel C-Clip
// Print in TPU 95A. Snaps over tongue-groove joint.
// ============================================================
module panel_clip(clip_h = 120) {
  total_w = clip_grip * 2 + clip_wall * 4;
  total_d = panel_t + clip_wall * 2;
  difference() {
    // Outer shell
    translate([0, 0, 0])
      cube([total_w, total_d, clip_h]);
    // Left channel
    translate([clip_wall, clip_wall, -1])
      cube([clip_grip + clip_wall, panel_t, clip_h + 2]);
    // Right channel
    translate([clip_grip + clip_wall * 3, clip_wall, -1])
      cube([clip_grip + clip_wall, panel_t, clip_h + 2]);
    // Round off bottom entry edges for easy snapping
    translate([clip_wall, clip_wall, 0])
      rotate([0, 0, 0])
        cylinder(h = 4, r = fillet_r);
  }
}

// ============================================================
// PART 3: Corner Connector (90°)
// Print in TPU 95A. Joins two perpendicular panels.
// ============================================================
module corner_connector(angle = 90, clip_h = 120) {
  arm = clip_grip + clip_wall * 2;
  total_d = panel_t + clip_wall * 2;
  difference() {
    union() {
      // Vertical arm
      cube([arm, total_d, clip_h]);
      // Horizontal arm
      rotate([0, 0, -(angle - 90)])
        translate([-total_d, 0, 0])
          cube([total_d, arm, clip_h]);
    }
    // Carve out vertical channel
    translate([clip_wall, clip_wall, -1])
      cube([clip_grip, panel_t, clip_h + 2]);
    // Carve out horizontal channel
    rotate([0, 0, -(angle - 90)])
      translate([-(total_d - clip_wall), clip_wall, -1])
        cube([clip_grip, panel_t, clip_h + 2]);
  }
}

// RENDER — uncomment the part you want to export as STL
// ============================================================
