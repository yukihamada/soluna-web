Problem

誰もいない家は、壊れていく。

空き家、別荘、民泊物件。年間300日以上、誰もいない。水漏れは数週間気づかない。火災は手遅れになるまで発見されない。侵入者は翌月のオーナー訪問まで分からない。

カメラは使わない。mmWaveレーダーは電波で存在だけを検知。映像なし。録音なし。プライバシーファースト。

How it Works · 仕組み

検知。通知。自動対処。

24/7監視。異常検知から通知まで数秒。全てNEUROダッシュボードで監視可能。

Water Leak
水漏れ検知 → 元栓自動閉止
水漏れセンサーが水を検知→即Push通知（LINE/メール）。オプションの電磁弁接続で元栓を自動閉止。被害をゼロに。
Intrusion
ドア開 + 人感 → サイレン + 通知
リードスイッチがドア開放を検知。mmWaveが人の存在を確認。サイレン鳴動 + オーナー通知 + 近隣通知。
Fire / Gas
煙/ガス検知 → 即時警報
MQ-2が煙・可燃ガスを検知。オーナーに即Push通知。初期段階での発見が全焼を防ぐ。
Freeze
水道管 <3℃ → ヒーター自動ON
DS18B20が水道管温度を常時監視。3℃以下でヒーター自動起動（リレー接続時）。または水抜き通知をオーナーに送信。
