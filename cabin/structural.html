02 / 08

積雪荷重 Snow Load

弟子屈町の設計積雪深120cmと設計用積雪密度200 kg/m³から、屋根積雪荷重を算定。片流れ屋根の形状係数による低減を適用する。

地上積雪荷重

S_ground = d x rho / 100
// d = 120 cm, rho = 200 kg/m³
S_ground = 120 x 200 / 100 = 240 kg/m²
積雪密度の根拠: 新雪30 kg/m³、締雪150 kg/m³、ざらめ雪300 kg/m³。北海道内陸部の長期堆積を考慮し、設計用平均密度として200 kg/m³を採用。

屋根形状係数

// 告示1455号（平成12年）片流れ・勾配 30° 以下
Cs = sqrt(cos(theta))
Cs = sqrt(cos(16.7°)) = sqrt(0.9578) = 0.9787

屋根設計積雪荷重

S_roof = Cs x S_ground
S_roof = 0.9787 x 240 = 235 kg/m²

屋根積雪荷重

235 kg/m²
地上240 kg/m² x Cs=0.979

積雪時 屋根総荷重

5,875 kg
235 x 25 m² = 57.6 kN
積雪荷重が支配荷重: 本建物では積雪235 kg/m²が屋根設計の主要因となる。風荷重・地震荷重より大きい。

03 / 08

風荷重 Wind Load

告示第1454号に基づく基準風速V₀=34 m/s、地表面粗度区分IIで速度圧を算定。転倒安定性と屋根吸い上げを検討する。

速度圧 q_H

q_H = 0.6 x E x V₀²

// 粗度区分II: Zb=5m, ZG=350m, alpha=0.15
// H=2.65m < Zb=5m → Er at H=Zb
Er = 1.7 x (5/350)^0.15 = 0.866
Gf = 2.2 // ガスト影響係数
E = Er² x Gf = 0.750 x 2.2 = 1.650

q_H = 0.6 x 1.650 x 34² = 1,145 N/m² (117 kgf/m²)

壁面風力係数

面Cf風圧 (N/m²)
風上壁（正圧）+0.8916
風下壁（負圧）-0.4458
側壁（負圧）-0.7802

転倒安定性

// 水平風力
F_wind = (0.8 + 0.4) x 1,145 x 13.25 m² = 18.2 kN

// 転倒モーメント
M_ot = 18.2 x 1.325 = 24.1 kN-m

// 安定モーメント
M_st = 3,165 x 9.81/1000 x 2.5 = 77.7 kN-m

SF = 77.7 / 24.1 = 3.22 ≥ 1.5

PASS 安全率 3.22 -- 転倒に対して十分安全

屋根吸い上げ

W_uplift = 1,145 x 25 = 28,625 N = 28.6 kN
W_roof = 1,125 x 9.81/1000 = 11.04 kN
// ネット吸い上げ
F_net = 28.6 - 11.04 = 17.6 kN
対策: 壁4面 x 各2箇所 = 計8箇所にHD金物。1箇所あたり 2.2 kN。HD-B10（許容10kN）で対応可。

PASS HD-B10金物で屋根吸い上げに対応

壁量計算 Wall Bracing

令第46条・告示第1352号に基づく必要壁量と、ハイブリッドSIPsパネルの存在壁量を比較し、耐震・耐風の充足を確認する。

必要壁量

// 地震時（多雪区域, 積雪120cm, 平屋）
基本係数 = 15 cm/m²
積雪割増 = 1.2 x 7 = 8.4 cm/m²
所要壁量 = 15 + 8.4 = 23.4 cm/m²

必要壁量(地震) = 25 x 23.4 = 585 cm = 5.85 m
// 風圧時（見付面積 x 50 cm/m²）
見付面積(1.35m超) = 5.0 x (2.65-1.35) + 1.25 = 7.75 m²
必要壁量(風圧) = 7.75 x 50 = 387.5 cm = 3.88 m
地震力が支配的: 各方向 5.85m 以上の壁量が必要。

壁倍率の推定

面材推定壁倍率根拠
杉CLT 12mm（外面）3.0倍構造用合板9mm=2.5倍ベース
竹集成 12mm（内面）4.0倍引張140MPa、せん断強度が杉の3倍以上
両面合算（保守値）5.0倍告示上限考慮の保守値

存在壁量（開口部を除く）

壁面全長開口部有効壁長
南面（正面）5.0 mドア0.8 + M窓x2 = 2.6m2.4 m
北面5.0 mS窓x2 = 1.2m3.8 m
東面5.0 mM窓x1 = 0.9m4.1 m
西面5.0 mM窓x1 = 0.9m4.1 m

壁量判定

方向必要壁量存在壁量充足率判定
X方向5.85 m 41.0 m (4.1+4.1)x5.0 701% PASS
Y方向5.85 m 31.0 m (2.4+3.8)x5.0 530% PASS

偏心率（4分割法）

南側/北側 壁量比 = (2.4 x 5.0) / (3.8 x 5.0) = 12.0 / 19.0 = 0.632 ≥ 0.5

PASS 4分割法 充足壁量比 0.632

開口余裕: 壁量は両方向とも大幅に充足。壁倍率を保守的に2.5倍（杉CLT面のみ）としても、X方向350%、Y方向265%で充足する。開口部をさらに増やす余地あり。

屋根構造 Roof Structure

カラマツ母屋90x90mmの曲げ応力とたわみを検討。積雪235 kg/m²の荷重下でスパンに対する安全性を確認する。

母屋の曲げ（スパン 2,500mm — 中間支持なし）

w = (45 + 235) x 0.91 = 254.8 kg/m = 2.498 kN/m

M = wL²/8 = 2.498 x 2.5² / 8 = 1.952 kN-m
Z = 90 x 90² / 6 = 121,500 mm³
sigma_b = 1.952 x 10⁶ / 121,500 = 16.1 MPa
等級許容曲げ fb判定
無等級材14.7 MPaNG
JAS E11022.2 MPaOK
目視1級20.4 MPaOK

母屋のたわみ（スパン 2,500mm）

delta = 5wL⁴ / (384EI)
= 5 x 2.498 x 2500⁴ / (384 x 11,000 x 5,467,500)
= 21.1 mm

L/300 = 2,500 / 300 = 8.3 mm

NG たわみ 21.1mm > 許容 8.3mm -- スパン2,500mmでは不可

対策が必要: 母屋90x90mmではスパン2,500mmで大きくたわむ。以下の対策案を検討。

対策案C（推奨）: 中間梁追加でスパン1,250mm

// スパン1,250mmに短縮
M = 2.498 x 1.25² / 8 = 0.488 kN-m
sigma_b = 0.488 x 10⁶ / 121,500 = 4.02 MPa < 14.7 MPa

delta = 5 x 2.498 x 1250⁴ / (384 x 11,000 x 5,467,500) = 1.32 mm
L/300 = 4.17 mm

PASS 曲げ 4.02 < 14.7 MPa（無等級OK）

PASS たわみ 1.32 < 4.17 mm

支圧チェック

R = wL/2 = 2.498 x 1.25 / 2 = 1.561 kN
sigma_c = 1,561 / (90 x 90) = 0.193 MPa < 2.4 MPa

PASS めり込み応力 十分安全

条件・注意事項

1. 母屋スパン: 90x90mmはスパン2,500mmでNG。中間梁を追加しスパン1,250mm以下にするか、断面を90x130mm以上に変更すること。
2. 母屋等級: スパン2,500mmの場合はJAS E110以上（or 目視1級以上）が必須。スパン1,250mmなら無等級でも可。
3. 壁倍率: 5.0倍は推定値。大臣認定取得が望ましい。認定前は保守的に2.5倍（杉CLT面のみ）で計算しても両方向で充足。
4. 地盤: 許容地耐力50 kN/m²は仮定値。SWS試験等で確認のこと。
5. 本書の位置づけ: 概算検討であり、確認申請用の構造計算書ではない。詳細設計には構造設計者による検討が必要。
