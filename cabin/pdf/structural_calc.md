## 3. 積雪荷重（Snow Load）

### 3.1 地上積雪荷重

- 弟子屈 設計積雪深: d = 120 cm = 1.2 m
- 積雪単位荷重（設計用平均密度）: ρ = 200 kg/m³
  - 根拠: 新雪30 kg/m³、締雪150 kg/m³、ざらめ雪300 kg/m³の平均
  - 北海道内陸部の長期堆積を考慮し200 kg/m³を採用

```
S_ground = d × ρ / 100
S_ground = 120 × 200 / 100
S_ground = 240 kg/m²
```

### 3.2 屋根積雪荷重

- 屋根勾配: θ = 16.7°（3寸勾配）
- 屋根形状係数: μ_b = cos^1.5(θ)（片流れ屋根, 令86条2項）

```
μ_b = cos^1.5(16.7°)
    = (0.9578)^1.5
    = 0.9365
```

- 但し、建基法では滑雪を考慮しない安全側の Cs（=1.0）を使う場合が多い。
- ここでは告示に従い、勾配による低減を適用:

```
Cs = √cos(θ) = √(0.9578) = 0.9787
```

※ 告示1455号（平成12年）の屋根形状係数。片流れ・勾配≤30°の場合。

```
S_roof = Cs × S_ground
       = 0.9787 × 240
       = 234.9 kg/m²
```

**設計積雪荷重: S_roof = 235 kg/m²**

### 3.3 積雪時の屋根総荷重

```
W_snow = S_roof × A_roof(水平投影)
       = 235 × 25
       = 5,875 kg
```

## 4. 風荷重（Wind Load）

### 4.1 基本パラメータ

- 基準風速: V₀ = 34 m/s（北海道, 告示1454号）
- 地表面粗度区分: II（田園・郊外）
- 基準高さ: H = 2.65 m（平均壁高）

### 4.2 速度圧 q_H

```
q_H = 0.6 × E × V₀²
```

地表面粗度区分IIの場合:
- E_r = 1.7 × (Z_b/Z_G)^α × (H/Z_b)^(2α) （H ≤ Z_b の場合）
- 粗度区分II: Z_b = 5m, Z_G = 350m, α = 0.15
- H = 2.65m < Z_b = 5m → E_r を H=Z_b=5m で評価

```
E_r = 1.7 × (5/350)^0.15 × 1.0
    = 1.7 × 0.5095
    = 0.866
```

ガスト影響係数 G_f（粗度区分II, H≤5m）:
```
G_f = 2.2（簡便法）
```

```
E = E_r² × G_f = 0.866² × 2.2 = 0.750 × 2.2 = 1.650
```

```
q_H = 0.6 × 1.650 × 34²
    = 0.6 × 1.650 × 1,156
    = 1,144 N/m²
    = 116.8 kgf/m²
```

**設計速度圧: q_H ≈ 1,145 N/m² (117 kgf/m²)**

### 4.3 風力係数

#### 壁面（風上・風下）

| 面 | 風力係数 C_f |
|----|------------|
| 風上壁（正圧） | +0.8 |
| 風下壁（負圧） | −0.4 |
| 側壁（負圧） | −0.7 |

壁面風圧:
```
W_wall = q_H × C_f
風上: 1,145 × 0.8 = 916 N/m² (93 kgf/m²)
風下: 1,145 × 0.4 = 458 N/m² (47 kgf/m²)
壁面合計圧: 916 + 458 = 1,374 N/m² (140 kgf/m²)
```

#### 屋根面（片流れ θ=16.7°）

| 風向 | 風上側 C_f | 風下側 C_f |
|------|----------|----------|
| 高→低（軒→棟方向） | −1.0 | −0.5 |
| 低→高（棟→軒方向） | +0.2 / −0.5 | −0.5 |
| 横風 | −0.8 | −0.8 |

最大吸い上げ:
```
W_uplift = q_H × |C_f_max|
         = 1,145 × 1.0
         = 1,145 N/m² (117 kgf/m²)
```

### 4.4 風力に対する転倒モーメント

最悪ケース: 風上壁 + 風下壁の合計圧を受ける

```
水平風力 F_wind = (C_f_windward + C_f_leeward) × q_H × A_wall
A_wall = 5.0 × 2.65 = 13.25 m²
F_wind = 1.2 × 1,145 × 13.25 = 18,205 N = 18.2 kN
```

転倒モーメント（基礎底面回り）:
```
M_ot = F_wind × H/2 = 18.2 × 1.325 = 24.1 kN·m
```

安定モーメント（建物自重による抵抗）:
```
M_st = W_dead × g × B/2
     = 3,165 × 9.81/1000 × 5.0/2
     = 31.06 × 2.5
     = 77.7 kN·m
```

安全率:
```
SF = M_st / M_ot = 77.7 / 24.1 = 3.22 > 1.5 ✓ OK
```

### 4.5 屋根吸い上げチェック

```
吸い上げ力 = 1,145 × 25 = 28,625 N = 28.6 kN
屋根自重 = 1,125 × 9.81/1000 = 11.04 kN
```

ネット吸い上げ = 28.6 − 11.04 = **17.6 kN（引き抜き方向）**

→ 屋根と壁の接合部にホールダウン金物が必要。
→ 壁4面、各面にHD金物2箇所 = 8箇所
→ 1箇所あたり: 17.6 / 8 = 2.2 kN = 224 kgf
→ **HD-B10（許容引張 10kN）で十分対応可。**

**判定: PASS**

```
Q_d = 0.2 × 1.0 × 1.0 × 1.0 × 51.2
```

## 6. 壁量計算（Wall Bracing Calculation）

### 6.1 必要壁量 — 地震力

令第46条・告示第1352号に基づく。

**地震時の必要壁量:**

平屋（多雪区域, 積雪120cm）:

```
必要壁量 = 床面積 × 所要壁量係数

基本係数（平屋）: 15 cm/m²
積雪割増: 積雪1mあたり +7 cm/m²（多雪区域軽い屋根）
割増 = 1.2 × 7 = 8.4 cm/m²

地震時所要壁量 = (15 + 8.4) = 23.4 cm/m²
必要壁量(地震) = 25 × 23.4 = 585 cm = 5.85 m
```

### 6.2 必要壁量 — 風圧力

```
見付面積 = 壁面面積(風方向)
X方向見付面積: 5.0 × 2.65 = 13.25 m²（屋根面含まず）
        屋根三角: 5.0 × 0.5 / 2 = 1.25 m²
        合計: 14.5 m²

風圧時の所要壁量 = 見付面積 × 所要係数
一般地域: 50 cm/m²

必要壁量(風圧, X方向) = 14.5 × 50 / 100 = 7.25 m ← 原式修正

実際: 令46条4項の表では:
50 cm/m² は多雪区域以外。
多雪区域の補正は風圧では不要（風圧は積雪と独立）。

必要壁量(風圧) = 見付面積 × 50 cm/m²
```

X方向:
```
見付面積 = 13.25 + 1.25 = 14.5 m²
1.0m以下の部分を除く: 14.5 − 5.0×1.0 = 9.5 m²（GL+1mから計算）
※実際は令46条の見付面積は「床面から1.35mを超える部分」
修正見付面積 = 5.0 × (2.65 − 1.35) + 1.25 = 5.0 × 1.30 + 1.25 = 7.75 m²

必要壁量(風圧, X方向) = 7.75 × 50 = 387.5 cm = 3.88 m
```

Y方向:
```
見付面積 = 5.0 × (2.65 − 1.35) + 5.0 × 0.25(勾配平均) = 6.5 + 1.25 = 7.75 m²
必要壁量(風圧, Y方向) = 7.75 × 50 = 387.5 cm = 3.88 m
```

### 6.3 必要壁量（決定値）

| 方向 | 地震時 | 風圧時 | 必要壁量（大きい方）|
|------|--------|--------|-------------------|
| X方向 | 5.85 m | 3.88 m | **5.85 m** |
| Y方向 | 5.85 m | 3.88 m | **5.85 m** |

**→ 地震力が支配的。各方向 5.85m 以上の壁量が必要。**

### 6.5 存在壁量の計算

各壁面の有効壁長（開口部を除く）:

| 壁面 | 全長 | 開口部 | 有効壁長 |
|------|------|--------|---------|


## 7. 屋根構造検討

### 7.1 母屋の曲げ検討

**条件:**
- 母屋: カラマツ 90×90mm
- スパン: 910mm（パネル幅方向の母屋間隔ではなく、母屋の支持スパン）
- ※ 注意: 母屋のスパンは桁間距離。片流れ屋根で桁行5m、中間に棟木・母屋あり。
- 実際の母屋スパン = パネル幅 5.0m を中間支持なしで架ける場合 → 要中間支持。
- ここでは棟木1本を中間（2.5m位置）に設け、母屋スパン L = 2,500mm と仮定。

荷重:
```
w = (D_roof + S_roof) × 母屋負担幅
  = (45 + 235) × 0.91
  = 280 × 0.91
  = 254.8 kg/m = 2.498 kN/m
```

曲げモーメント（単純梁）:
```
M = wL²/8
  = 2.498 × 2.5² / 8
  = 2.498 × 6.25 / 8
  = 1.952 kN·m
```

断面係数:
```
Z = bh²/6 = 90 × 90² / 6 = 121,500 mm³ = 1.215 × 10⁻⁴ m³
```

曲げ応力:
```
σ_b = M / Z = 1.952 × 10⁶ / 121,500 = 16.07 N/mm² = 16.1 MPa
```

カラマツの許容曲げ応力度:
- 無等級材: f_b = 14.7 MPa（建基法告示）
- 機械等級 E110: f_b = 22.2 MPa
- JAS目視等級1級: f_b = 20.4 MPa

```
σ_b = 16.1 MPa > 14.7 MPa（無等級） → NG
σ_b = 16.1 MPa < 22.2 MPa（E110）  → OK
σ_b = 16.1 MPa < 20.4 MPa（1級）   → OK
```

**判定: 要JAS等級材（E110以上 or 目視1級以上）。無等級材では NG。**

### 7.2 母屋のたわみ検討

```
δ = 5wL⁴ / (384EI)
```

- E（カラマツ E110）= 11,000 MPa = 11,000 N/mm²
- I = bh³/12 = 90 × 90³ / 12 = 5,467,500 mm⁴

```
δ = 5 × 2.498 × 2,500⁴ / (384 × 11,000 × 5,467,500)
  = 5 × 2.498 × 3.906 × 10¹³ / (384 × 11,000 × 5,467,500)

単位統一（N, mm）:
w = 2.498 N/mm
L = 2,500 mm

δ = 5 × 2.498 × 2500⁴ / (384 × 11,000 × 5,467,500)
  = 5 × 2.498 × 3.906×10¹³ / (384 × 6.014×10¹⁰)
  = 4.879×10¹⁴ / (2.309×10¹³)
  = 21.1 mm
```

たわみ制限:
```
L/300 = 2,500 / 300 = 8.3 mm
δ = 21.1 mm > 8.3 mm → NG
```

**判定: NG — スパン2,500mmでは母屋90×90はたわみ過大。**

### 7.3 対策案

**案A: 母屋断面を大きくする**
```
必要I = 5wL⁴ / (384 × E × δ_allow)
      = 5 × 2.498 × 2500⁴ / (384 × 11,000 × 8.33)
      = 4.879×10¹⁴ / (3.518×10¹⁰)
      = 13,870,000 mm⁴

90×h: I = 90h³/12 → h = ∛(13,870,000×12/90) = ∛(1,849,333) = 122.7mm

→ 90×130mm 以上が必要（I = 90×130³/12 = 16,462,500 mm⁴ > 13,870,000 ✓）
```

**案B: 中間支持を増やす（スパンを短くする）**
```
スパン1,250mm（支持点3箇所）の場合:
δ = 5 × 2.498 × 1250⁴ / (384 × 11,000 × 5,467,500)
  = 5 × 2.498 × 2.441×10¹² / (2.309×10¹³)
  = 1.32 mm
L/300 = 4.17 mm
1.32 < 4.17 → OK ✓

曲げ: M = wL²/8 = 2.498 × 1.25² / 8 = 0.488 kN·m
σ_b = 0.488×10⁶ / 121,500 = 4.02 MPa < 14.7 MPa → OK ✓
```

**案C: 90×90mm母屋 + 中間に梁1本追加（スパン1,250mm）→ PASS**

**推奨: 案C — 棟方向に中間梁を1本追加し母屋スパンを1,250mmに短縮。**

### 7.4 支持点の支圧チェック

母屋端部反力:
```
R = wL/2 = 2.498 × 1.25 / 2 = 1.561 kN（案Cの場合）
```

支圧面積（母屋90mm幅 × 座掘り90mm）:
```
A_bear = 90 × 90 = 8,100 mm²
σ_c⊥ = 1,561 / 8,100 = 0.193 MPa
```

カラマツ めり込み許容応力: f_c⊥ = 2.4 MPa（無等級でも）

```
0.193 MPa << 2.4 MPa → OK ✓
```

**判定: PASS**

## 8. 基礎検討

### 8.1 設計条件

- 基礎形式: RC布基礎
- 凍結深度: 1.2 m（弟子屈）
- 根入れ深さ: D_f = 1.2 m
- 地盤: 火山灰質土（弟子屈は摩周カルデラ周辺の火山灰台地）
- 許容地耐力: q_a = 50 kN/m²（長期, 仮定）

### 8.2 基礎幅の検討

布基礎の底版幅 B を求める。

建物荷重（長期）:
```
W_total = W_dead + W_floor_live + W_snow(長期)
W_dead = 3,165 kgf = 31.05 kN
W_floor_live = 180 × 25 × (60/180) = 130 kgf/m²(長期低減) × 25 = 3,250 kgf = 31.88 kN
  ※令85条: 住宅床の長期低減なし → 180 × 25 = 4,500 kgf = 44.15 kN

多雪区域の長期積雪荷重:
W_snow_long = 0.7 × S × A = 0.7 × 235 × 25 = 4,112.5 kgf = 40.34 kN
  ※0.7は多雪区域の長期積雪荷重低減係数

W_total = 31.05 + 44.15 + 40.34 = 115.54 kN
```

布基礎の総延長:
```
外周: 4 × 5.0 = 20.0 m
内部基礎梁: 1本 × 5.0m = 5.0 m（仮定）
基礎総延長 L_f = 25.0 m
```

基礎底面の接地圧:
```
q = W_total / (B × L_f)
```

必要基礎幅:
```
B ≥ W_total / (q_a × L_f)
B ≥ 115.54 / (50 × 25.0)
B ≥ 0.092 m = 92 mm
```

**最小基礎幅 B = 150 mm（布基礎の最小値として）**

但し、令第38条・告示第1347号より:
- 布基礎の底版幅: 450mm 以上（告示仕様規定）
- 立上り厚: 120mm 以上
- 立上り高: GL+300mm 以上

**設計値: B = 450mm（告示仕様規定に従う）**

### 8.3 接地圧チェック

```
q = 115.54 / (0.45 × 25.0)
  = 115.54 / 11.25
  = 10.27 kN/m²
```

```
q = 10.27 kN/m² < q_a = 50 kN/m² → OK ✓
充足率: 10.27 / 50 = 20.5%
```

**判定: PASS（大幅に余裕あり）**
