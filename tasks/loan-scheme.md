## ⚠️ 最重要：先に確認すること（不特法）

SOLUNAは「共有持分を販売し、Airbnb賃料収益を各オーナーに分配する」。
これは**不動産特定共同事業法（不特法）第2条の「不動産特定共同事業」に該当する可能性が高い**。

```
不特法 第2条：
「不動産を賃貸し、その収益を分配することを目的として
複数の者から出資を受けることを業として行うもの」
```

### 不特法に該当する場合の要件
| 種別 | 要件 |
|------|------|
| 第1号事業（自己取引型） | 純資産額1億円以上・国交大臣許可 |
| 第3号事業（小規模特例） | 純資産1,000万円以上・都道府県知事届出 |
| 適格特例投資家等限定事業 | プロ投資家向けの場合 |

**アクション**: 国土交通省（地方整備局）への事前相談を最優先で実施。
既に運営中であれば現行スキームのコンプライアンス確認も同時に。

## 法的根拠マップ

```
不動産（共有持分）の割賦販売に関係する法律：

宅建業法 §43-45  ← メイン規律（宅建業者の割賦販売）
     ↓
割賦販売法         ← 不動産は「指定商品」に含まれない → 非適用
     ↓
貸金業法           ← 「売買代金の後払い」≠「金銭の貸付け」→ 非適用
                    （スキーム設計を誤ると適用される可能性あり）
     ↓
不特法             ← 賃料収益分配があれば要確認（最重要）
     ↓
利息制限法         ← 分割手数料を金利として取る場合：上限年18%
```

## 推奨スキーム：宅建業者自身の割賦販売

### 資金フロー

```
【買主】
  ↓ Step1: 頭金（物件価格の30%以上）
【Enabler Inc.】 → Step2: 持分移転登記実行 → 【買主】が登記オーナーに
  ↓ Step3: 残金（70%以下）を12〜60回払い
【Stripe Billing】 が毎月自動引落し → Enabler入金
  ↓ Step4 (万一): 支払い停止 → 30日書面催告 → 解除 → 持分抹消訴訟
```

### 宅建業法の制約と対応

| 条文 | 規制内容 | 対応策 |
|------|----------|--------|
| §44（30%ルール） | 登記前に受領できるのは代金の30%まで | 頭金30%を受け取ったら即座に登記移転 |
| §43（解除制限） | 支払遅滞でも30日書面催告なしに解除不可 | 自動催告フロー（Stripe失敗→翌日通知→30日後解除） |
| §45 | 割賦条件を重要事項説明書に記載義務 | 重説書テンプレート改訂 |

### 価格帯別 分割プラン例

| 物件 | 購入価格 | 頭金（30%） | 残金 | 月払い×60回 | 実質年率 |
|------|---------|------------|------|------------|---------|
| ロッジ | ¥490万 | ¥147万 | ¥343万 | ¥6.2万/月 | 2.0% |
| NESTING | ¥980万 | ¥294万 | ¥686万 | ¥12.4万/月 | 2.0% |
| OFF-GRID | ¥890万 | ¥267万 | ¥623万 | ¥11.3万/月 | 2.0% |
| TAPKOP | ¥3,900万 | ¥1,170万 | ¥2,730万 | ¥49.4万/月 | 2.0% |

※ 年利2.0%（利息制限法の上限は年18%。2%は消費者向けとして現実的）
※ 割賦手数料は「分割払い手数料」として別途表示（貸金業法回避のため「利息」表記は避ける）

## 回収不能リスクの管理

### リスクシナリオと対応

```
買主が支払い停止
    ↓
Stripe失敗通知 → システムが自動メール送信（翌日）
    ↓ 7日以内に解決しない場合
書面（内容証明）による30日催告（宅建業法§43）
    ↓ 30日後も不払い
契約解除の通知 → 弁護士依頼
    ↓
持分移転登記の抹消請求訴訟（地方裁判所）
    ↓
勝訴→登記抹消→持分回収→再販売
```

### リスク軽減策（契約書に盛り込む）

1. **共有物分割禁止特約の登記**（民法§256）
   - 5年ごと更新の共有物分割禁止を登記
   - 買主が持分を勝手に分割・処分するのを防止

2. **代理受領特約**
   - Airbnb収益の分配金を自動的に割賦金に充当する特約
   - 実質的に売掛金と相殺できる構造

3. **利用権停止条項**
   - 2ヶ月以上の不払い時はオーナーダッシュボードのアクセスを停止
   - 法的解除とは別に事実上のプレッシャー

4. **緊急連絡先の取得**
   - 保証人ではなく、緊急連絡先（家族）の登録を任意で求める

## 将来的な拡張オプション

### オプションA: 頭金10%スキーム（リスク高）
- §44対応のため「頭金10%+引渡し後分割払い」の設計
- 技術的には可能だが、回収不能リスクが大幅増
- 不動産市況が良い物件（TAPKOP等）に限定適用を検討

### オプションB: 自社ローン会社設立（3年後）
- 貸金業登録（純資産5,000万円以上が目安）
- 担保権行使が可能になり、回収リスクが大幅低下
- 金利収益が追加収益源に
- 今は時期尚早、実績を積んでから

### オプションC: 提携ノンバンク（検討）
- 共有持分専門のローン提供会社は少ないが、交渉の余地あり
- アイフルビジネスファイナンス、ダイヤモンドアセット等に打診
- 成功すれば最もリスクフリー
