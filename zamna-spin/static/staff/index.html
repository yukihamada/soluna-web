STAFF MANUAL

ZAMNA HAWAII 2026 — Operations Guide for All Staff & Volunteers

Roles & Staffing

Gate Staff

8–10 ppl

Ticket scanning, ID verification (21+), bag check assist, wristband application

VIP Host

4–6 ppl

VIP & Diamond guest concierge, access control, hospitality

Stage Crew

6–8 ppl

Stage changeover, equipment management, artist stage access support

F&B Staff

15–20 ppl

Bar operations (3 bars), food booths, alcohol ID check, inventory

Security

30–40 ppl

Access control, patrol, emergency response, misconduct handling (outsourced)

Medical Team

3–4 ppl

EMTs, nurse, medical tent operations

Shuttle / Traffic

4–6 ppl

Shuttle guidance, rideshare zone management, parking direction

Cleanup / Sustainability

6–8 ppl

Recycling station management, waste sorting, post-event teardown

Media / PR Attend

2 ppl

Press check-in, media guidance, social media photo support

Total: approx. 80–110 staff (incl. security)

Shift Schedule

Setup Shift

06:00–18:00

Venue setup, sound check, catering load-in

Event Shift A

16:00–03:30

Pre-open to close. Gate, VIP, F&B, security

Event Shift B

20:00–03:30

Peak hours reinforcement. Extra F&B, security

Teardown Shift

03:00–08:00

Cleanup, equipment removal, waste sorting

What to Bring

  • •Wear black T-shirt (staff shirts provided on-site)
  • •Comfortable, closed-toe shoes (safety shoes recommended, no heels)
  • •Sunscreen & hat (for daytime shifts)
  • •Water bottle (staff hydration station available)
  • •Fully charged mobile phone
  • •Photo ID / driver's license
  • •If applicable: personal medication, allergy info card

Code of Conduct

  • •Zero alcohol on duty. Immediate termination if caught
  • •Mobile phones: break time only. Silent mode during shifts
  • •Treat all guests with courtesy and professionalism
  • •Report issues to your lead via radio first. Do not act alone
  • •Breaks in designated area (backstage rest room) only. No eating in venue
  • •No social media posts about event content or artists (NDA signed)
  • •In emergencies, follow safety plan. Guest evacuation is top priority

Radio Channels

CH 1Operations (All)
CH 2Security
CH 3Stage / Sound
CH 4F&B / Catering
CH 5Medical / Emergency
CH 6Shuttle / Traffic

Emergency Quick Reference

Injured / ill guest found

→ Call medical on CH5 → Guide to medical tent

Suspicious person / misconduct

→ Report to security on CH2 → Stand clear, observe

Evacuation order (PA announcement)

→ Guide guests to nearest exit → Stay calm, no running

Fire / smoke

→ Call 911 + report on CH1 → Evacuate guests
