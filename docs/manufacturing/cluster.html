SECTION 01

なぜ建築確認が不要か

CLUSTERは3つの法的根拠を組み合わせることで、建築確認申請なしに有機形状ユニット群を実現する。各ユニットが独立した構造体として成立し、デッキで接続することで実質無制限の居住空間を生み出す。

建築基準法 第6条 — 確認申請の要否

  • 都市計画区域外 + 床面積10m²以下 → 建築確認申請不要（法第6条第2項）
  • 弟子屈・美留和地区 = 都市計画区域外（非線引き区域） → 第1要件を満たす
  • スクリューパイル基礎 = 地盤に定着しない → 仮設的扱い可能（建告第1347号）
  • 各ユニット独立 = 「増築」でなく「別棟」 → 各棟単独で10m²判定（合算なし）

実質的上限なし — 面積の積み上げ

  • ユニット1棟: 9.9m²（確認不要）
  • ユニット5棟: 49.5m²（各棟独立で判定）
  • デッキ（屋根なし）: 「建築物」に該当しない → 面積に算入せず
  • 5棟 + デッキ60m²: 実質 約110m² の生活空間
注意事項: 確認申請は不要でも、建築基準法の技術基準（採光・換気・構造強度）は適用されます。また、農地転用・農振法の手続きは別途必要な場合があります。スクリューパイルの仮設的扱いは施工方法によって異なるため、施工前に専門家への確認を推奨します。本ページの内容は一般的な解釈に基づくものです。

有機形状がもたらすもの

直方体のキャビンと違い、有機形状は「生き物が作った巣」のような佇まいを持つ。菌糸体成形の特性（型に従いながら生長する）は有機的な曲面と相性が抜群。標準的な木造在来工法では難しい形状が、菌糸体パネルなら比較的低コストで実現できる。各ユニットが独立した個性を持ちながら、デッキで接続することで統一感のあるクラスターを形成する。

木造在来工法では実現困難な形状 菌糸体成形で低コスト実現 各棟9.9m²以下で建確不要 段階的にユニット追加可能

菌糸体成形の優位性 — なぜ有機形状に向いているか

菌糸体は液状の基材（おが粉 + 種菌）を型枠に充填し、生物的な成長力で型枠内を完全充填する。この特性はコンクリートのような流し込み材料と同様で、曲面型枠や複雑形状への適応が容易。木材やパネル材では製造が困難・高コストになる有機形状パネルが、菌糸体なら型枠コストのみで実現できる。型枠を一度作れば何度でも同じパネルを成形できるため、ユニット追加時もコスト変動がほぼない。

型枠に沿って完全充填 曲面・有機形状が得意 型枠再利用で量産低コスト 軽量 / 廃棄時コンポスト可能
