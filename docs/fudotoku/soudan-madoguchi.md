**グレーの理由**: SOLUNAは「売買」形式（法務局登記）を取っているが、

現時点で即座に「違法」とは断言できないが、**国交省に事前確認するまでOFF-GRID・TAPKOPの新規募集は止める**のが安全。

**自社でOK・他社と組む？**  
→ ロッジ・NESTINGは**今すぐ北海道知事に届出**を出せば自社で合法化。  
→ OFF-GRID・TAPKOPは**第1号許可が取れるまでは許可業者（不動産CRファンド等）と組む**のが最速。  
→ 自社で全部やるなら第1号許可申請（本書類一式）を進める。

## STEP 1: 国土交通省への事前相談（最優先・今週中）

### 相談先
```
国土交通省 土地・建設産業局 不動産業課
〒100-8918 東京都千代田区霞が関2-1-3
電話: 03-5253-8111（代表）→ 不動産業課へ転送
直通: 03-5253-8359（不動産業課 不動産特定共同事業担当）
```

### 相談の方法
- **事前予約制**（電話またはメールで予約）
- **匿名相談可**（会社名・事業者名を出さずに概要だけ相談可能）
- **費用無料**
- 持参資料：事業概要のA4一枚（物件概要・スキーム・価格・口数）

### 相談時に確認すること（この質問を持っていく）
1. 「共有持分の売買 + Airbnb管理委託 + 賃料分配」は不特法の「不動産特定共同事業」に該当するか
2. 全口合計1億円未満の物件（ロッジ・NESTING）は小規模特例（第3号）で届出のみでよいか
3. 第1号許可申請の際、設立直後の会社で純資産1億円要件の達成方法（増資・VC出資等）は認められるか
4. OFF-GRIDが「建設前物件」として募集することの留意点
5. 申請から許可まで通常何ヶ月かかるか（補正含む）

### 地方整備局への相談でも可
北海道内のみで事業を行う場合は都道府県知事許可（今回は複数都道府県の投資家を想定するため国交大臣許可が適切）。

## STEP 5: 第1号許可申請（国土交通大臣）

### 申請先（窓口）
```
国土交通省 土地・建設産業局 不動産業課
〒100-8918 東京都千代田区霞が関2-1-3
（持参の場合は事前予約制）
```

### 申請費用
収入印紙（金額は要確認。概ね数万円〜十数万円）

### 審査期間
申請受理から**概ね3〜6ヶ月**（補正がなければ。補正が入ると延長）。

### 許可要件のうち今すぐ対応が必要なもの

| 要件 | 現状 | アクション |
|------|------|----------|
| 純資産1億円以上 | East Ventures 5,000万円出資 → **不足** | **増資が必要**。追加VC調達またはデット調達で1億円まで積み上げ |
| 宅地建物取引業免許 | 状況確認要 | 未取得なら東京都知事免許を先に申請 |
| 業務管理者（宅建士+不特法講習） | 状況確認要 | 宅建士証の有効期限確認 + 不特法業務管理者講習受講 |
| 欠格事由なし | 全役員で確認 | 誓約書・身分証明書・住民票を準備 |
| 事務所の設置 | 東京都内に実態ある事務所 | 確認・整備 |

### 純資産1億円への道筋（重要）

East Ventures出資後の想定純資産が5,000万円台とすると、あと4〜5,000万円が必要。
選択肢：
1. **追加VC調達**: Series Aで1億円調達 → 自動的に達成
2. **不動産担保ローン**: 既存物件を担保に借入 → 自己資本は変わらないが事業継続資金に
3. **クラウドファンディング（株式型）**: FUNDINNOなどで小口調達
4. **創業者貸付の資本化**: 代表者貸付がある場合、資本金に組み入れ

## STEP 6: 業務管理者講習の受講

不特法第12条に基づく「業務管理者」は以下の要件を全て満たす必要がある：

### 要件
1. 宅地建物取引士の資格（有効な宅建士証）
2. **不動産特定共同事業業務管理者講習** の修了

### 講習実施機関
```
一般社団法人 不動産特定共同事業協会（不特協）
〒105-0001 東京都港区虎ノ門1-1-21
電話: 03-3508-8711
URL: https://www.fudotoku.or.jp/
```

### 講習の概要
- **内容**: 不特法の基礎・契約約款・財務管理・コンプライアンス等（約2日間）
- **受講料**: 約3〜5万円
- **開催**: 年数回（東京・大阪）
- **修了証**: 有効期限なし（ただし定期的な研修参加推奨）
