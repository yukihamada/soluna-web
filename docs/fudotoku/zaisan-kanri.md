## 1. 分別管理の基本方針

### 1-1. 法的根拠

株式会社Enabler（以下「当社」という）は、不動産特定共同事業法（以下「不特法」という）第15条および同法施行規則第22条の規定に従い、不動産特定共同事業に係る財産を当社固有の財産および他の事業に係る財産から厳格に分別して管理する。

### 1-2. 分別管理の対象財産

本事業において分別管理の対象となる財産は以下のとおりとする。

| 区分 | 内容 |
|------|------|
| 出資金（募集資金） | 投資家（共有持分購入者）からの出資金または共有持分購入代金 |
| 賃料収入 | 対象物件の宿泊・賃貸に係る収入（Airbnb等プラットフォーム経由含む） |
| 修繕積立金 | オーナー全員から徴収する修繕・維持管理費の積立金 |
| 固定資産税等納税準備金 | 固定資産税・都市計画税・損害保険料等の充当準備金 |
| 分配準備金 | オーナーへの利益分配前に一時的に保管する資金 |

### 1-3. 基本原則

1. **完全分離の原則**: 上記財産はいかなる場合も当社の運転資金・営業経費口座と混同しない。
2. **専用口座の設置**: 財産区分ごとに専用の銀行口座を設け、入出金を管理する。
3. **帳簿の独立**: 本事業専用の会計帳簿（勘定科目）を設け、月次で財務状況を把握する。
4. **透明性の確保**: オーナー（投資家）に対し、定期的に財産状況を報告する。

## 2. 出資金（募集資金）の管理方法

### 2-1. 専用口座の設置

当社は、本事業の出資金を管理するため、以下の専用口座を設置する。

| 項目 | 内容 |
|------|------|
| 口座名義 | 株式会社Enabler 不特事業出資金口 |
| 金融機関 | [銀行名・支店名を記入]（預金保険制度の対象機関を選定） |
| 口座種別 | 普通預金 |
| 管理責任者 | 代表取締役 濱田 優貴（および業務管理者 [氏名を記入]） |

### 2-2. 出資金の入金フロー

```
投資家（出資）
    ↓ 振込
[出資金専用口座]（当社名義・分別管理）
    ↓ 物件取得・設備投資に充当
対象不動産の取得・整備
```

### 2-3. 出資金の使途制限

出資金は以下の用途にのみ充当し、他の用途への流用を厳禁する。

- 対象不動産（土地・建物）の取得費用
- 建物改修・設備導入費用（オフグリッド設備含む）
- 登記費用・各種許認可取得費用
- 不動産取得税・印紙税等の公租公課
- 許可申請に係る弁護士・行政書士等の専門家費用

### 2-4. 信託活用の検討

資金規模・投資家人数が一定水準を超える場合、当社は信託銀行等に対して出資金の管理を委託する信託契約の締結を検討する。信託を利用する場合は、以下の条件を満たす機関を選定する。

- 信託業法に基づく免許を有する信託銀行または信託会社
- 投資家を受益者とする不動産管理信託または金銭信託
- 当社破綻時においても信託財産として保全される仕組み

## 3. 賃料収入の管理方法

### 3-1. 入金フローの概要

```
宿泊者
    ↓ 宿泊料支払い
Airbnb / その他OTA（予約プラットフォーム）
    ↓ 精算（売上確定翌月末を目安）
[賃料収入専用口座]（当社名義・分別管理）
    ↓ 費用控除後
[分配準備口座]
    ↓ 四半期ごと分配
各オーナー（持分比率に応じ）
```

### 3-2. 賃料収入専用口座

| 項目 | 内容 |
|------|------|
| 口座名義 | 株式会社Enabler 不特事業賃料収入口 |
| 金融機関 | [銀行名・支店名を記入] |
| 管理責任者 | 業務管理者 [氏名を記入] |

### 3-3. 費用控除の対象と優先順位

賃料収入から控除する費用の優先順位は以下のとおりとする。

1. OTAプラットフォーム手数料（Airbnb等の徴収手数料）
2. 清掃費・リネン交換費等の直接運営費
3. 管理委託手数料（当社または外部管理会社への報酬）
4. 修繕積立金への拠出（月次定額）
5. 固定資産税等納税準備金への拠出（年税額÷12）
6. 損害保険料の按分積立
7. 当社の事業報酬（事業約款所定の料率）

上記控除後の残額を分配対象利益とする。

### 3-4. 分配サイクル

- **四半期分配**: 各四半期（3月・6月・9月・12月の末日）を基準日とし、基準日から30日以内に各オーナーの指定口座へ振込分配を行う。
- **持分比率による按分**: 各オーナーの共有持分割合に基づき按分する。
- **最低分配金額**: [金額を記入]円未満の場合は翌四半期へ繰り越す。

## 4. 修繕積立金の管理

### 4-1. 修繕積立金の目的

対象物件の長期的な価値維持および突発的な修繕費用への備えとして、修繕積立金を設ける。

### 4-2. 積立方法

| 区分 | 積立方法 | 積立額の目安 |
|------|---------|------------|
| 月次積立（賃料収入から拠出） | 毎月の賃料収入の[X]%を自動拠出 | 月額[金額を記入]円を上限 |
| 初期一括積立 | 出資金から一定額を初期積立として拠出 | [金額を記入]円 |

### 4-3. 修繕積立金専用口座

| 項目 | 内容 |
|------|------|
| 口座名義 | 株式会社Enabler 不特事業修繕積立口 |
| 金融機関 | [銀行名・支店名を記入] |
| 管理方法 | 定期預金または元本保証型商品による運用を原則とする |

### 4-4. 修繕積立金の使途

修繕積立金は以下の用途に限定して使用する。

- 建物の外装・屋根・基礎等の大規模修繕
- オフグリッド設備（太陽光発電・蓄電池・浄水設備等）の更新・修繕
- 給排水・電気・暖房設備の更新
- 緊急修繕（台風・大雪等による突発的被害の復旧）

### 4-5. 修繕計画

当社は、物件取得後5年・10年・15年を目安とした長期修繕計画書を作成し、オーナーへ開示する。修繕計画は専門家（建築士等）の意見を踏まえ3年ごとに見直す。

## 5. 固定資産税等の納税管理

### 5-1. 対象税目

| 税目 | 納付時期 | 管理方法 |
|------|---------|---------|
| 固定資産税 | 年4回（4月・7月・12月・翌2月） | 納税準備金口座から充当 |
| 都市計画税 | 固定資産税と合算 | 同上 |
| 損害保険料 | 契約更新時（年1回） | 修繕積立金または納税準備金から充当 |

### 5-2. 納税準備金口座

賃料収入専用口座から月次で積立を行い、納期到来時に充当する。共有不動産の固定資産税は原則として持分割合に応じて按分し、当社が一括納付する。

### 5-3. 税負担の開示

各オーナーへの年次報告において、持分に対応する固定資産税等の負担額を明示する。

## 6. 会計・監査体制

### 6-1. 会計帳簿の作成

当社は、不特法第25条の規定に基づき、本事業専用の会計帳簿を作成・保存する。

| 帳簿種類 | 保存期間 | 担当 |
|---------|---------|------|
| 現金出納帳（口座入出金明細） | 10年 | 経理担当者 |
| 賃料収支台帳 | 10年 | 業務管理者 |
| 修繕積立金管理台帳 | 10年 | 業務管理者 |
| 分配計算書 | 10年 | 経理担当者 |

### 6-2. 会計基準

本事業の会計処理は、一般的に公正妥当と認められる企業会計の慣行に従う。外部監査の要否については、投資家人数・事業規模に応じて検討し、必要な場合は公認会計士または監査法人に依頼する。

### 6-3. 内部監査

代表取締役は、半期に1度（毎年6月・12月）、業務管理者による財産管理状況の自主点検（内部監査）を実施させ、その結果を記録する。

### 6-4. 外部専門家との連携

- **税理士**: 確定申告・税務対応
- **弁護士**: 契約書・法的リスク対応
- **公認会計士**: 財務諸表の確認・外部監査（必要に応じ）

## 7. 財産状況の報告方法（オーナーへの開示）

### 7-1. 定期報告の種類

| 報告種類 | 頻度 | 内容 | 方法 |
|---------|------|------|------|
| 四半期報告書 | 年4回 | 賃料収入・費用明細・分配金額・修繕積立金残高 | 電子メール・専用ウェブポータル |
| 年次報告書 | 年1回（3月末） | 年間収支報告・財産状況・税務関連情報（確定申告用） | 電子メール・書面（希望者） |
| 臨時報告 | 随時 | 大規模修繕・天災等の重要事象発生時 | 電子メール・電話 |

### 7-2. 開示項目（四半期報告書）

1. 対象物件の稼働状況（稼働率・稼働日数・宿泊者数）
2. 賃料収入の総額および内訳（OTA別）
3. 控除費用の明細（清掃費・管理費・積立拠出額等）
4. 当期分配対象利益
5. 各オーナーへの分配金額（持分別）
6. 修繕積立金の残高および当期拠出額
7. 物件の主要な変更事項・修繕実施状況

### 7-3. 開示プラットフォーム

当社は、オーナーが24時間いつでも財産状況を確認できる専用ウェブポータル（以下「オーナーポータル」という）を提供する予定である。同ポータルでは、報告書のダウンロード、過去実績のグラフ閲覧、分配履歴の確認が可能とする。

### 7-4. 報告書の保存

当社は、全ての報告書を作成後10年間保存し、オーナーおよび監督官庁（国土交通大臣）から求めがあった場合は速やかに提示する。
